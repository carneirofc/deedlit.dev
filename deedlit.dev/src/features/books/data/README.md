# Books Configuration Guide

This system is designed to minimize manual work when managing your book collection. Just add Goodreads URLs and let the scraper handle the rest!

## Quick Start

### Adding a New Book

1. **Open** [books-config.ts](./books-config.ts)
2. **Add** the Goodreads URL to the `bookUrls` array:
   ```typescript
   export const bookUrls = [
     "https://www.goodreads.com/book/show/6683140-color-and-light",
     "https://www.goodreads.com/book/show/YOUR_NEW_BOOK_ID",  // ← Add here
   ];
   ```
3. **Run** the scraper:
   ```bash
   npm run scrape-goodreads
   ```
4. **Done!** The metadata is automatically fetched and your app is updated.

## File Structure

### 📄 Generated by Scraper
- **`books-metadata.json`** - Auto-generated book metadata from Goodreads
  - ⚠️ Don't edit this file manually - it's overwritten by the scraper

### ⚙️ Configuration Files
- **`books-config.ts`** - Your book URLs and custom overrides
  - Add URLs here to manage your collection
  - Add custom notes, tags, or year overrides

### 📚 App Files
- **`books.ts`** - Loads and merges metadata with overrides
  - Automatically used by the app
  - No manual editing needed

## Adding Custom Metadata

Sometimes you want to add personal notes or tags. Use `bookOverrides`:

```typescript
export const bookOverrides: Record<string, Partial<{
  year: number;
  note: string;
  tags: string[];
}>> = {
  "https://www.goodreads.com/book/show/6683140-color-and-light": {
    year: 2010,
    note: "Lighting, palette control, atmosphere.",
    tags: ["color-theory", "painting", "visual-arts", "realism"],
  },
};
```

### What can be overridden?
- **`year`** - Publication year (if scraper gets it wrong)
- **`note`** - Short personal note about the book
- **`tags`** - Custom tags for filtering (supplements scraped genres)

## Workflow

### Simple Workflow (Recommended)
1. Add Goodreads URL to `books-config.ts`
2. Run `npm run scrape-goodreads`
3. View your updated collection

### Advanced Workflow
1. Add URL to `books-config.ts`
2. (Optional) Add overrides for personal notes/tags
3. Run `npm run scrape-goodreads`
4. Review generated `books-metadata.json`
5. Adjust overrides if needed

## What Gets Scraped?

The scraper fetches:
- ✅ Title
- ✅ Author
- ✅ Rating (average)
- ✅ Ratings count
- ✅ Reviews count
- ✅ Genres
- ✅ Page count
- ✅ Description
- ✅ Cover image URL
- ✅ Publisher (when available)

## Troubleshooting

### Scraper returns empty data
Goodreads has anti-scraping measures. If this happens:
1. Check if the URLs are correct
2. Wait a few minutes and try again
3. Manually add data to `books-metadata.json` as a fallback

### Book not appearing in the app
1. Check that the URL is in `bookUrls` array
2. Run `npm run scrape-goodreads` to regenerate metadata
3. Restart your dev server (`npm run dev`)

### Want to remove a book
1. Remove the URL from `books-config.ts`
2. Run `npm run scrape-goodreads`
3. The metadata file will be regenerated without it

```bash
npm run scrape-goodreads
```

That's it! Your books are now configured and displayed in the app with filtering, sorting, and all metadata automatically handled.
